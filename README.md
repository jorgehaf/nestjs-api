## Installation

```bash
$ npm install
```

## Running the app

```bash
# create image
$ npm run image

# running the app in http://localhost:3000
$ npm run container

```

## Endpoints doc

```bash
# get version API
$ curl --location 'http://localhost:3000/about'

# post login with token return
$ curl --location 'http://localhost:3000/login/teste@abc.com'

```
